Данные подтягиваются из массива products (наименования, id, цены для каждого цвета). Цены и id установлены мною рандомно.
Товары, у которых задан oldprice автоматически помечаются меткой "sale".
По клику на изображении в правом блоке происходит выбор товара (выводится в консоль) и рендеринг цен для текущего варианта товара и цвета.
По клику на "Add to cart" отправляется request и возвразается response (выводится в консоль).
Адаптивность выполнена частично, т.к. макет для смартфона в ТЗ отсутствует.
